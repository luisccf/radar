Furto é uma ação caracterizada pela ausência de violência ou grave ameaça contra uma pessoa. Por isso, se o criminoso ameaçou (com ou sem o uso de armas) ou agrediu a vítima para subtrair-lhe os pertences, não é possível solicitar o B.O. pela internet. Neste caso, procure a delegacia de polícia mais próxima.
